# KHOJ-A Research Funding Tool Using Blockchain

<img src="KHOJ-HomePage.png">

## Run Locally

To run this project, you will need to add the deployed contract address to src/abi/contractaddress.json.

```bash
  npm i
  cd Khoj
  npm start
```
